
  # Gamification Strategy Dashboard UI

This project is a modern Gamification Strategy Dashboard UI designed in Figma to enhance user engagement through game-inspired experiences. It features interactive elements such as challenges, achievements, leaderboards, rewards, XP tracking, and progress analytics in a clean, user-friendly interface. The design follows modern UI/UX principles with a responsive layout, intuitive navigation, and a premium visual style, making it suitable for web and SaaS applications.  

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
